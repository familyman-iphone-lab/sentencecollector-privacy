문장수집가 개인정보처리방침

최종 수정일: 2026년 5월 1일 · 시행일: 2026년 5월 1일

문장수집가(Sentence Collector)는 이용자의 개인정보를 어떠한 형태로도 수집하지 않습니다. 모든 데이터는 이용자의 기기 내부에만 저장됩니다.

1. 개인정보 수집 여부

문장수집가는 이용자의 개인정보를 일체 수집하지 않습니다. 본 앱은 별도의 회원가입이나 로그인 절차가 없으며, 이용자의 이름, 이메일, 전화번호, 위치, 연락처 등 어떠한 식별 가능한 정보도 요구하지 않습니다.

2. 데이터 저장 방식

이용자가 앱에 입력하는 모든 데이터는 오직 이용자 본인의 기기 내부 저장소(SwiftData)에만 저장됩니다.

저장되는 데이터 저장 위치 외부 전송 여부
수집한 문장 본문 기기 내부 전송하지 않음
책 정보(제목, 저자, 출판사) 기기 내부 전송하지 않음
태그 및 색상 마커 기기 내부 전송하지 않음
개인 메모 및 감상 기기 내부 전송하지 않음
책 표지 이미지 기기 내부 전송하지 않음
OCR로 추출한 문장의 원본 사진(선택) 기기 내부 전송하지 않음
알림 설정값 기기 내부 전송하지 않음

3. 권한 사용

문장수집가는 다음의 기기 권한을 요청할 수 있으며, 각 권한의 사용 목적은 아래와 같습니다.

권한 사용 목적 거부 시 영향
알림(Notifications) 일일 문장 푸시 알림 알림 기능 사용 불가, 다른 기능은 정상 사용 가능
카메라(Camera) 책 사진을 촬영하여 OCR로 문장을 추출(Premium) 카메라 OCR 사용 불가, 텍스트 직접 입력 및 갤러리 선택은 가능
사진(Photos) - 읽기 갤러리의 책 사진을 선택하여 OCR로 문장 추출(Premium) 갤러리 OCR 사용 불가, 카메라 촬영 및 텍스트 입력은 가능
사진(Photos) - 추가 생성한 문장 카드 이미지를 갤러리에 저장(Premium) 이미지 저장 불가, 공유 시트를 통한 공유는 가능

이러한 권한을 통해 접근한 데이터는 모두 기기 내부에만 저장되며, 외부로 전송되지 않습니다.

4. OCR(광학 문자 인식) 처리 방식

OCR 기능은 Apple의 Vision 프레임워크를 사용하며, 모든 인식 처리는 이용자의 기기 내에서만 수행됩니다. 사진이나 인식 결과가 외부 서버나 클라우드로 전송되지 않습니다.
  1. 이용자가 카메라로 촬영하거나 갤러리에서 선택한 사진은 기기 내 Vision 프레임워크를 통해 텍스트가 추출됩니다.
  2. 추출된 텍스트 중 이용자가 선택한 부분만 문장으로 저장되며, 선택하지 않은 텍스트는 폐기됩니다.
  3. 이용자가 "원본 사진 보존" 옵션을 선택한 경우, 해당 사진은 기기 내부에만 저장되며 외부로 전송되지 않습니다.

5. 결제 정보

  1. 문장수집가의 Premium 결제는 Apple App Store의 In-App Purchase 시스템을 통해 처리됩니다.
  2. 개발자는 이용자의 신용카드 번호, 결제 수단 등 어떠한 결제 관련 정보에도 접근하지 않습니다.
  3. 결제 처리 및 영수증 발행은 전적으로 Apple이 담당하며, 이에 대한 자세한 사항은 Apple Media Services 이용약관을 참조해 주시기 바랍니다.

6. 제3자 제공 및 위탁

개발자는 이용자의 개인정보를 제3자에게 제공하거나 외부에 위탁하지 않습니다. 본 앱은 다음의 외부 서비스를 사용하지 않습니다:

  • 분석 도구(Google Analytics, Firebase Analytics 등)
  • 광고 네트워크
  • 크래시 리포팅 도구
  • 외부 클라우드 백업 서비스
  • 외부 OCR API 또는 AI 서비스
유일한 외부 연동은 Apple의 In-App Purchase(StoreKit 2)이며, 이는 Apple이 직접 처리하는 결제 시스템입니다. OCR 처리도 Apple의 Vision 프레임워크를 통해 기기 내에서만 이루어집니다.

7. 데이터 보관 및 삭제

  1. 이용자가 앱에 입력한 데이터는 이용자가 직접 삭제하거나 앱을 삭제할 때까지 기기 내부에 보관됩니다.
  2. 앱을 삭제하면 모든 데이터가 즉시 영구 삭제되며, 복구할 수 없습니다.
  3. 중요한 데이터는 Premium의 JSON 백업 또는 CSV 내보내기 기능을 통해 별도로 백업하실 것을 권장합니다.

8. 공유 기능과 개인정보

  1. 이용자가 문장 카드 이미지를 외부 SNS, 메신저 등으로 공유하는 경우, 해당 공유는 이용자의 자발적 행위이며 개발자는 이에 관여하지 않습니다.
  2. 공유된 콘텐츠는 해당 플랫폼의 정책에 따라 처리되므로, 이용자는 공유 전 해당 플랫폼의 개인정보처리방침을 확인하시기 바랍니다.

9. 아동의 개인정보

문장수집가는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 본 앱은 데이터 수집 자체가 없으므로 모든 연령대에서 안전하게 사용 가능합니다.

10. 이용자의 권리

이용자는 자신의 데이터에 대해 다음의 권리를 가집니다:

  • 열람: 앱 내에서 본인의 모든 데이터를 자유롭게 열람할 수 있습니다.
  • 수정: 앱 내에서 언제든지 본인의 데이터를 수정할 수 있습니다.
  • 삭제: 앱 내에서 개별 데이터를 삭제하거나, 앱 자체를 삭제하여 모든 데이터를 일괄 삭제할 수 있습니다.
  • 이동: Premium의 JSON / CSV / Markdown 내보내기 기능을 통해 본인의 데이터를 외부로 이동할 수 있습니다.
